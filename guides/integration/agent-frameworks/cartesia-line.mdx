## Prerequisites

- The agent ID from Cartesia of the agent you wish to use
- [An access token from Cartesia with appropriate agent grants](https://docs.cartesia.ai/get-started/authenticate-your-client-applications)

### Prepare your ElevenLabs agent provider details

Now that you have your agent details, take a note of the following snippet:

```typescript
type VoiceAgentDetails = {
  type: 'cartesia';
  agent_id?: '<your_cartesia_agent_id>;
  token?: '<your_cartesia_access_token>;
};
```

We'll be using this snippet in the steps to follow.
