Keyframe personas integrate with both agent frameworks (e.g., ElevenLabs, Cartesia) and real-time LLMs (e.g., Gemini Live, OpenAI Realtime). While each integration has minor setup differences, the overall flow is largely the same.

## Pipeline overview

<Steps>
  <Step title="Bring your own LLM or agent">
    Your client application continues to interface with your existing agent framework or live model API
  </Step>
  <Step title="Keyframe no-code or low-code components">
    Recieves "raw" audio from your agent or model, and relays it to the Keyframe platform
  </Step>
  <Step title="Rendered persona">
    The Keyframe components also manage rendering the generated persona in your clientapplication
  </Step>
</Steps>

```
+---------------------+                                     +--------------------------+
|                     | ---------- User input ------------> |                          |
|                     |                                     |    Your existing agent   |
|                     | <--- Raw agent / model audio ------ |         or model         |
|                     |       (will now be "relayed")       |                          |
|                     |                                     +--------------------------+
|   Your client app   |
|                     |                                     +--------------------------+
|                     | ------ Relayed agent audio -------> |                          |
|                     |                                     |         Keyframe         |
|                     | <------ Rendered persona ---------- |         platform         |
|                     |    (w/ synced audio and video)      |                          |
+---------------------+                                     +--------------------------+
```
