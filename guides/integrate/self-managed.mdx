title: "Self managed"
description: "Best if you want maximum control"

A self managed pipeline allows you to bring your own agent or LLM and keep your API keys on your own infrastructure. Build a fully custom embed using the same JS primitives that power our Widget (no-code) and Hosted (low-code) approaches.

For self managed pipelines, Keyframe personas integrate with both agent frameworks (e.g., Cartesia, ElevenLabs, LiveKit, Vapi) and real-time LLMs (e.g., Gemini Live, OpenAI Realtime). While each integration has minor setup differences, the overall flow is largely the same.

## Pipeline overview

<Steps>
  <Step title="Bring your own agent or LLM">
    Your client application continues to interface with your existing agent framework or real-time LLM
  </Step>
  <Step title="Keyframe low-code components">
    Our [easy-to-use components](/sdk-reference/javascript) receive "raw" audio from your agent or LLM, and relays it to the Keyframe platform
  </Step>
  <Step title="Rendered persona">
    The Keyframe components also manage rendering the generated persona in your client application
  </Step>
</Steps>

```text
+---------------------+                                     +--------------------------+
|                     | ----------- User input -----------> |                          |
|                     |                                     |    Your existing agent   |
|                     | <----- Raw agent / LLM audio ------ |         or LLM           |
|                     |       (will now be "relayed")       |                          |
|                     |                                     +--------------------------+
|   Your client app   |
|                     |                                     +--------------------------+
|                     | ---- Relayed agent / LLM audio ---> |                          |
|                     |                                     |         Keyframe         |
|                     | <------- Rendered persona --------- |         platform         |
|                     |    (w/ synced audio and video)      |                          |
+---------------------+                                     +--------------------------+
```
