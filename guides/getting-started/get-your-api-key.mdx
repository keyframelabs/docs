---

Your API key is used to authenticate your requests to the Keyframe API and is required for integrating the personas you create into your own applications. API keys are managed via the [Keyframe platform dashboard](https://platform.keyframelabs.com).

## Create a new API key

From the [API keys page](https://platform.keyframelabs.com/api-keys), click on the `Create new key` button.

![Create New Key](/images/create-new-key.png)

Click `Create key` and your new API key will sjhown to you. Remember to save it somewhere safe as you will not be able to access it later.

![Copy New Key](/images/copy-new-key.png)
