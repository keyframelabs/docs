<Info>
  To use a persona in an API call, set the `persona_id` or `persona_slug` field to the listed ID or slug.
</Info>

<Tip>
  Browse all available personas and try them live in the [Keyframe platform dashboard](https://platform.keyframelabs.com/persona-library).
</Tip>

## persona-1.5-live

<Info>
  The `persona-1.5-live` model is our first that is capable of expressing emotion. When integrated with a [supported agent / LLM provider](/guides/integration/integration-capability-matrix), the persona will express anger, happiness and sadness.
</Info>
